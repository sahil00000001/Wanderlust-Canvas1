## Packages
framer-motion | Page transitions, scroll-triggered animations, and micro-interactions

## Notes
- Fully static frontend application, no backend API calls needed.
- Uses hardcoded mock data for all travel packages, destinations, and testimonials.
- Relies on Unsplash for high-quality travel imagery.
- Uses CSS variables in index.css to override the default theme with a custom luxurious travel palette (Sky Blue, Ocean Teal, Sunset Orange).
- Expects 'Outfit' and 'DM Sans' fonts to be applied via CSS variables.
